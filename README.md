# billpleaseapi
>Server API backend application

## How to run
1. Clone repository and _cd_ there
2. ```npm install```
3. ```npm start```
4. Test requests (by Postman or curl) on **localhost:3000** 

### Routes for model *

| Method        | Path             | Params           |
| ------------- |:----------------:| :---------------:|
| **POST**      | /image           | _image_ {String} |
